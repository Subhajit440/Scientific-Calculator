# Scientific-Calculator
Developing a Scientific Calculator using HTML,CSS,JS 
Scientific Calculator

Overview

This project is a Scientific Calculator built using HTML, CSS, and JavaScript. The calculator is designed to perform a wide range of mathematical operations, from basic arithmetic to advanced scientific functions. Its clean and intuitive interface ensures that users can perform calculations seamlessly.

Features

Basic Arithmetic Operations: Addition, subtraction, multiplication, and division.

Advanced Mathematical Functions:

Trigonometric functions (sin, cos, tan, etc.)

Logarithmic calculations (log, ln)

Exponentiation and square roots


Memory Functions: Save, recall, and clear memory.

Responsive Design: Works across all screen sizes, including mobile devices.

Keyboard Support: Perform calculations using keyboard input
